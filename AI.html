2. Semantic Grouping (Intent)

Categorizes data based on meaning and intent, a key step before statistical analysis.

Details: Logic vs. Statistics
Semantic Grouping (Manual/Logical) gathers keywords by topic (e.g., 'solar kit' and 'solar panel' are Solar/PV). This differs from Clustering, which groups by metrics (e.g., all high-CTR keywords).

üß≠ Prompting Guide: How to Interact for Perfect Output

The accuracy of AI visualization depends on the clarity of your input. Follow these 4 steps to guide the analysis.

1. GOAL

Define the Analytical Objective

What do you want to discover? (Comparison, Distribution, Relationship, Trend). AI uses this to select the base chart type.

Example: "I want to analyze the Relationship between CTR and PR variables."

2. VARIABLES

Name the Variables and Their Role

Which data to use? Specify the columns (X and Y) and whether one is the dependent or independent variable.

Example: "Use CTR on the X-axis and PR on the Y-axis."

3. AI METHOD

Request the Specific ML Technique

If the goal is advanced, state the technique (Clustering, Scaling, Regression, etc.) and the parameters.

Example: "Apply Standard Scaling and then K-Means with $K=3$."

4. FORMAT

Define the Output and Style

Specify the output (Chart, Table), the visual style (colors, title), or the narration request (NLG).

Example: "Return a Scatter Plot colored by cluster. Provide an NLG summary."
